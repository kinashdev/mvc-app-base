## Controladores
### Como usar?
Basta adicionar isso na sua **index.php**:

`use Application\Controllers as Controller;`

`$controller = new Controllers\Controller('nome_de_usuario');`

**OBS:** sem o `nome_de_usuario` não é possível usar os Controladores;
